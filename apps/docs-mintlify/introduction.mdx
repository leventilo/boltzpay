---
title: BoltzPay
description: Open-source SDK that gives AI agents a fetch() that pays.
---

BoltzPay is an open-source TypeScript SDK that lets AI agents pay for API data automatically. One `fetch()` call handles protocol detection, payment signing, and settlement. Your agent gets the data, the server gets paid.
